# 空间聚焦算子（A6）技术文档

## 1. 在系统中的定位
空间聚焦算子负责把“看整帧”转化为“看局部对象或区域”。它是从全局视频理解走向对象级、区域级证据建模的关键模块。

## 2. 文献依赖等级

- 文献依赖等级：`P2 论文原生`
- 最新支撑强度：`很强`

### 2.1 文献锚点
- VideoRefer Suite
- SpaceVLLM
- CAT-V
- VideoLoom
- SlotVTG

### 2.2 从文献保留了什么
- 对象级区域是高质量时空 grounding 的核心入口
- 空间定位不应该只是辅助显示，而应当成为独立证据
- 区域信息需要与时间信息一起被后续算子消费

## 3. 子模式定义

### 3.1 BBOX
输出目标框或显著性框。

### 3.2 MASK
输出实例或区域 mask。

### 3.3 SALIENT_CROP
输出局部裁剪图，供下游标注或 OCR 使用。

### 3.4 SOM_MARK
输出带编号或标记的区域图，增强对象级引用能力。

## 4. 输入规范
消费 `samples[]`，追加 `regions[]`。

### 4.1 必需输入
- `sample_id`
- `resource_refs.frame_uri`

### 4.2 可选输入
- `config.region_mode`
- `config.max_regions_per_frame`
- `config.target_classes`
- `config.min_box_area_ratio`

## 5. 输出规范
每个局部区域写入 `regions[]`，推荐字段：

- `region_id`
- `sample_id`
- `region_type`
- `bbox`
- `mask_uri`
- `crop_uri`
- `mark_id`
- `confidence`

## 6. 关键参数

| 参数 | 作用 |
|---|---|
| `max_regions_per_frame` | 控制每帧保留区域数 |
| `target_classes` | 指定关注对象类型 |
| `min_box_area_ratio` | 过滤过小区域 |
| `mark_style` | SoM 标记样式 |

## 7. 与上下游的绑定关系
- 上游：A3/A4 提供代表帧
- 下游：A8 可生成对象级 caption 和空间 QA
- 下游：A7 可结合局部对象变化做时序摘要

## 8. 典型失败模式
- 框太多，造成视觉噪声
- 框太少，漏掉关键对象
- 局部裁剪脱离上下文，影响语义理解

## 9. 推荐使用场景
- 多对象交互视频
- 指代类、定位类 QA
- 需要对象级时间跟踪或局部 OCR 的场景
