# 算子-论文映射

## 1. 使用说明
本文件用于回答“某个算子为什么合理”和“它在现有文献中是否有明确支撑”这两个问题。

映射规则如下：

- `P2`：可以直接对齐到论文中的核心模块或明确处理环节
- `P1`：由多篇论文中的共同需求抽象出来
- `P0`：系统闭环必需，但不是论文中的显式独立模块

## 4. 不同等级的算子定义

### P2 算子
该算子的核心能力已经被近期文献反复验证，当前的系统是在做可复用的原子化整理。

### P1 算子
该级别的算子并非某一篇论文的原生模块，但它稳定抽取了多篇论文中的共同需求。

### P0 算子
主要用于保证工程闭环和输入纯度，其必要性来自系统运行，而不是源于论文。
