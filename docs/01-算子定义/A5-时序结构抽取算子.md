# A5-时序结构抽取算子

## 1. 当前实现

| 项目 | 内容 |
|---|---|
| 注册名 | `video_temporal_structure` |
| 实现类 | `VideoTemporalStructureOperation` |
| 源码 | `AscendDataForge/multimodal_data_processor/operations/augmentation/video_temporal_structure_operation.py` |
| 基类 | `VideoAugmentationOperation` |
| operation_type | `OperationType.AUGMENTATION` |

A5 根据 `segments`、`samples` 以及可选的 `description_units`、`text_signals` 构建 `temporal_structures`。

## 2. 支持模式

| 模式 | 行为 | 依赖 |
|---|---|---|
| `EVENT_WINDOW` | 每个 segment 生成一个 event window，最多 `max_temporal_units` 个 | 不调用语言模型 |
| `EVENT_CHAIN` | 对连续 segment 窗口生成事件链摘要 | 调用 `transformers` 因果语言模型 |

当前没有 `MULTIHOP_BRIDGE` 模式。

## 3. 输入字段

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `segments_key` | `segments` | 必需 |
| `samples_key` | `samples` | 必需 |
| `description_units_key` | `description_units` | 可选 |
| `text_signals_key` | `text_signals` | 可选 |

`execute()` 会要求 `segments` 和 `samples` 非空。

## 4. 输出字段

A5 写回：

- `temporal_structures`
- `temporal_result`
- `temporal_mode`
- `temporal_metrics`

每个 temporal structure 包含：

- `id`
- `temporal_unit_id`
- `segment_id`
- `segment_ids`
- `timestamp_range.start_sec`
- `timestamp_range.end_sec`
- `structure_type`：`event_window` 或 `event_chain`
- `summary`
- `precedence_refs`

`temporal_result.global_metrics` 包含：

- `total_segments`
- `total_samples`
- `total_temporal_structures`

## 5. 参数

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `output_key` | `temporal_structures` | 输出字段 |
| `result_key` | `temporal_result` | 完整结果字段 |
| `temporal_mode` | `EVENT_WINDOW` | `EVENT_WINDOW` 或 `EVENT_CHAIN` |
| `max_temporal_units` | `12` | 最多输出结构数量 |
| `chain_window_size` | `3` | `EVENT_CHAIN` 连续窗口大小，实际至少为 2 |
| `model_path` | `Qwen/Qwen2.5-0.5B-Instruct` | `EVENT_CHAIN` 使用的模型 |
| `max_new_tokens` | `96` | 事件链摘要最大生成 token 数 |

## 6. 行为细节

- `EVENT_WINDOW`：按 `segments[:max_temporal_units]` 逐段构造。
- `EVENT_CHAIN`：从第二个 segment 开始滑动构造窗口，直到达到 `max_temporal_units`。
- 摘要优先使用 `description_units`，其次使用 `text_signals`，再次使用 sample 数量作为描述；三者都缺失时会触发 warning。

## 7. 最小示例

```python
from multimodal_data_processor.core.operation_registry import OperationRegistry

op = OperationRegistry.create_operation(
    "video_temporal_structure",
    {"temporal_mode": "EVENT_WINDOW", "max_temporal_units": 12},
)
result_dataset = op.execute(dataset)
```
