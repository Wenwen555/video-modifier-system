# 文本辅证抽取算子（A6）

## 1. 当前实现

| 项目 | 内容 |
|---|---|
| 注册名 | `video_text_evidence` |
| 实现类 | `VideoTextEvidenceOperation` |
| 源码 | `AscendDataForge/multimodal_data_processor/operations/augmentation/video_text_evidence_operation.py` |
| 基类 | `VideoAugmentationOperation` |
| operation_type | `OperationType.AUGMENTATION` |

A6 从采样帧、字幕文件或 ASR 文件生成 `text_signals`。默认模式是 OCR，并默认使用 Hugging Face TrOCR 模型。

## 2. 支持模式

| 模式 | 行为 | 依赖 |
|---|---|---|
| `OCR` | 对 `samples[].frame_path` 运行 OCR；或在关闭 HF OCR 时读取 `samples[].ocr_text` | 默认需要 `transformers`、`torch`、`Pillow` 和帧图片 |
| `SUBTITLE_PARSE` | 从 `row["video"]["subtitle_uri"]` 读取 JSON/JSONL 文本条目并按 segment 时间重叠对齐 | 本地字幕 JSON/JSONL |
| `ASR_ALIGN` | 从 `row["video"]["asr_uri"]` 读取 JSON/JSONL 文本条目并按 segment 时间重叠对齐 | 本地 ASR JSON/JSONL |

## 3. 输入字段

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `segments_key` | `segments` | 必需 |
| `samples_key` | `samples` | 必需 |
| `video_key` | `video` | 字幕/ASR URI 容器 |
| `frame_path_key` | `frame_path` | OCR 图片路径字段 |
| `ocr_text_key` | `ocr_text` | `use_huggingface_ocr=False` 时读取 |

`execute()` 会要求 `segments` 和 `samples` 非空。`SUBTITLE_PARSE` 和 `ASR_ALIGN` 使用的文本文件需包含条目字段：`start_sec`、可选 `end_sec`、`text`、可选 `language`。

## 4. 输出字段

A6 写回：

- `text_signals`
- `text_evidence_result`
- `text_evidence_mode`
- `text_evidence_metrics`

每个 text signal 包含：

- `id`
- `text_signal_id`
- `signal_id`
- `segment_id`
- `timestamp_range.start_sec`
- `timestamp_range.end_sec`
- `text_type`
- `text`
- `language`

`text_evidence_result.global_metrics` 包含：

- `total_segments`
- `total_samples`
- `total_text_signals`

## 5. 参数

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `output_key` | `text_signals` | 输出字段 |
| `result_key` | `text_evidence_result` | 完整结果字段 |
| `text_evidence_mode` | `OCR` | `OCR`、`SUBTITLE_PARSE`、`ASR_ALIGN` |
| `merge_strategy` | `BY_SEGMENT` | `KEEP_ALL` 或 `BY_SEGMENT` |
| `default_language` | `zh` | 缺省语言 |
| `use_huggingface_ocr` | `True` | 是否运行 TrOCR |
| `model_name` | `trocr_small_printed` | `trocr_small_printed` 或 `trocr_base_printed` |
| `frame_path_key` | `frame_path` | sample 图片路径字段 |
| `ocr_text_key` | `ocr_text` | 预提取 OCR 文本字段 |

## 6. 合并策略

- `KEEP_ALL`：保留每条原始文本信号。
- `BY_SEGMENT`：按 `(segment_id, text_type, language)` 合并，扩大时间范围并拼接不重复文本。

## 7. 最小示例

```python
from multimodal_data_processor.core.operation_registry import OperationRegistry

op = OperationRegistry.create_operation(
    "video_text_evidence",
    {"text_evidence_mode": "OCR", "use_huggingface_ocr": False},
)
result_dataset = op.execute(dataset)
```
