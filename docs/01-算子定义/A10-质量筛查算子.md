# 质量筛查算子（A10）

## 1. 当前实现

| 项目 | 内容 |
|---|---|
| 注册名 | `video_quality_screening` |
| 实现类 | `VideoQualityScreeningOperation` |
| 源码 | `AscendDataForge/multimodal_data_processor/operations/filtering/video_quality_screening_operation.py` |
| 基类 | `FilteringOperation` |
| operation_type | `OperationType.FILTERING` |

A10 读取 `qae_triplets` 和可选 `revisions`，在内存中应用 revision 后生成 `quality_records`。它是当前 A 编号主链的最终质量记录生成算子。

## 2. 支持模式

| 模式 | 行为 |
|---|---|
| `RULE_FILTER` | 缺少 question/answer 或缺少 evidence refs 时输出 `drop`，否则 `keep` |
| `TIER_ASSIGN` | 按 evidence 数量和 `grounding_level` 分配数据层级，默认不因缺证据直接丢弃 |

当前没有 `SCORE_GATE` 或 `VALUE_RANK` 模式。

## 3. 输入字段

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `qae_triplets_key` | `qae_triplets` | 必需 |
| `revisions_key` | `revisions` | 可选 |

`execute()` 会要求 `qae_triplets` 非空。

## 4. 输出字段

A10 写回：

- `quality_records`
- `quality_result`
- `quality_mode`
- `quality_metrics`

每条 quality record 包含：

- `id`
- `quality_id`
- `target_triplet_id`
- `question`
- `quality_mode`
- `scores.evidence_score`
- `scores.completeness_score`
- `decision`：`keep` 或 `drop`
- `reason`
- `rank_in_video`
- `assigned_dataset_tier`

`quality_result.global_metrics` 包含：

- `total_triplets`
- `total_quality_records`

## 5. 分数与层级

A10 当前使用两项分数：

- `evidence_score`：`evidence_refs` 中有效引用数量。
- `completeness_score`：question 和 answer 都存在时为 `1.0`，否则为 `0.0`。

层级分配规则：

| 条件 | `assigned_dataset_tier` |
|---|---|
| `evidence_score > 1` 或 `grounding_level == "strong_grounded"` | `strong_grounded_qa` |
| `evidence_score == 1` | `grounded_qa` |
| 其他 | `qa` |

如果设置 `required_dataset_tier` 且分配结果不匹配，decision 会变成 `drop`。

## 6. 参数

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `output_key` | `quality_records` | 输出字段 |
| `result_key` | `quality_result` | 完整结果字段 |
| `quality_mode` | `RULE_FILTER` | `RULE_FILTER` 或 `TIER_ASSIGN` |
| `required_dataset_tier` | `None` | 要求的数据层级 |

## 7. 最小示例

```python
from multimodal_data_processor.core.operation_registry import OperationRegistry

op = OperationRegistry.create_operation(
    "video_quality_screening",
    {"quality_mode": "RULE_FILTER", "required_dataset_tier": "strong_grounded_qa"},
)
result_dataset = op.execute(dataset)
```
