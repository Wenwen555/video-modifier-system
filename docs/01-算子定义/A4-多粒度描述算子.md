# A4-多粒度描述算子

## 1. 当前实现

| 项目 | 内容 |
|---|---|
| 注册名 | `video_multigranularity_description` |
| 实现类 | `VideoMultigranularityDescriptionOperation` |
| 源码 | `AscendDataForge/multimodal_data_processor/operations/augmentation/video_multigranularity_description_operation.py` |
| 基类 | `VideoAugmentationOperation` |
| operation_type | `OperationType.AUGMENTATION` |

A4 使用图像 caption 模型对 A3 采样帧生成描述，并按 frame、event、segment 或 context 粒度写入 `description_units`。

## 2. 支持模式

| 模式 | 行为 | 必需输入 |
|---|---|---|
| `FRAME_DESCRIPTION` | 对每个 sample 生成一条 frame 描述 | `samples[].frame_path` |
| `EVENT_DESCRIPTION` | 对每个 temporal structure 覆盖的 samples 生成 event 描述 | `temporal_structures` + `samples[].frame_path` |
| `SEGMENT_DESCRIPTION` | 对每个 segment 的 samples 合并生成 segment 描述 | `segments` + `samples[].frame_path` |
| `CONTEXT_DESCRIPTION` | 按 `segments[].stitch_group_id` 分组生成 context 描述 | `segments[].stitch_group_id` + `samples[].frame_path` |

当前没有 `MULTI_SENTENCE_SUMMARY` 模式。

## 3. 模型

`model_name` 支持：

- `blip_base` -> `Salesforce/blip-image-captioning-base`
- `blip_large` -> `Salesforce/blip-image-captioning-large`
- `git_base` -> `microsoft/git-base-coco`
- `vit_gpt2` -> `nlpconnect/vit-gpt2-image-captioning`

`model_path` 为空时按 `model_name` 解析。运行需要 `transformers`、`torch` 和 `Pillow`。

## 4. 输入字段

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `segments_key` | `segments` | 输入分段字段 |
| `samples_key` | `samples` | 输入采样字段 |
| `temporal_structures_key` | `temporal_structures` | `EVENT_DESCRIPTION` 使用 |
| `frame_path_key` | `frame_path` | sample 中的图片路径字段 |

`execute()` 会要求 `segments` 和 `samples` 非空。所有实际 caption 都会读取 `sample[frame_path_key]`。

## 5. 输出字段

A4 写回：

- `description_units`
- `description_result`
- `description_mode`
- `description_metrics`

每个 description unit 包含：

- `id`
- `description_id`
- `description_unit_id`
- `segment_id`
- `description_text`
- `description_scope`
- `granularity`
- `anchor_refs`
- `summary_tags`

`description_result.global_metrics` 包含：

- `total_segments`
- `total_samples`
- `total_description_units`

## 6. 参数

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `output_key` | `description_units` | 输出描述单元字段 |
| `result_key` | `description_result` | 完整结果字段 |
| `description_mode` | `SEGMENT_DESCRIPTION` | 描述粒度模式 |
| `model_name` | `blip_base` | 模型别名 |
| `model_path` | `None` | 自定义模型路径 |
| `max_new_tokens` | `64` | 最大生成 token 数 |
| `frame_path_key` | `frame_path` | 图片路径字段 |

## 7. 最小示例

```python
from multimodal_data_processor.core.operation_registry import OperationRegistry

op = OperationRegistry.create_operation(
    "video_multigranularity_description",
    {"description_mode": "SEGMENT_DESCRIPTION", "model_name": "blip_base"},
)
result_dataset = op.execute(dataset)
```
