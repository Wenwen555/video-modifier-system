# 预设管线与文献对照

## 1. 文档目标

本文件把当前已实现的视频算子组合成可说明、可落地的预设管线。这里的“可落地”指算子注册名、模式和字段依赖均能在当前 AscendDataForge 源码中找到。

## 2. 当前默认 preset

当前实现最直接的 structure-first preset：

```text
video_partition
  -> video_context_orchestration
  -> video_sampling
  -> video_multigranularity_description
  -> video_temporal_structure
  -> video_text_evidence
  -> video_scaffold_construction
  -> video_qae_materialization
  -> video_consistency_revision
  -> video_quality_screening
```

对应 A 编号：

`A1 -> A2 -> A3 -> A4 -> A5 -> A6 -> A7 -> A8 -> A9 -> A10`

注意：若 A4 使用 `EVENT_DESCRIPTION`，需要先运行 A5 生成 `temporal_structures`；若 A6 使用默认 OCR，A3 通常需要 `save_sample_frames=True`。

## 3. 系统级 preset

| 预设名称 | 可落地链路 | 关键配置 | 输出 |
|---|---|---|---|
| `Structure-First Preset` | `A1 -> A2 -> A3 -> A4/A5/A6 -> A7 -> A8 -> A9 -> A10` | A7 使用 `EVENT_TREE` 或 `ACTION_GRAPH`；A8 设置 `use_llm_client` | `quality_records` |
| `Narration-First Preset` | `A1 -> A2 -> A3 -> A6 -> A5 -> A7 -> A8 -> A9 -> A10` | A6 使用 `SUBTITLE_PARSE` 或 `ASR_ALIGN`；A7 使用 `NARRATION_CHUNK` | narration scaffold 驱动的 `qae_triplets` |
| `QA-First Grounding Preset` | `外部 qa_drafts + A1/A2/A3/A4/A5/A6 -> 已有 scaffold_units -> A8 -> A9 -> A10` | A8 使用 `QA_FIRST_GROUNDING`；scaffold 需含 `source_qa_draft_ids` | grounded `qae_triplets` |

当前 A7 没有 `QUERY_SCAFFOLD` 模式，因此 QA-first preset 不能写成“A7 自动从 QA 草稿构造 query scaffold”。

## 4. 文献对照总表

| 文献工作 | 路径归类 | 当前实现映射 | 说明 |
|---|---|---|---|
| `Panda-70M` | 基础切分/拼接 | A1 `FIXED/SHOT` + A2 `MERGE_STITCH` | 对应 split-and-stitch 的工程化子集 |
| `TimeChat-Captioner` | 时间结构与描述 | A3 + A4 + A5 + A6 | 当前 A5 支持 event window/chain，不支持独立 multihop bridge |
| `Grounded Question-Answering in Long Egocentric Videos` | narration-first | A6 + A5 + A7 `NARRATION_CHUNK` + A8 | 需要字幕/ASR/narration 输入 |
| `Grounded Multi-Hop VideoQA in Long-Form Egocentric Videos` | structure-first | A5 + A7 `ACTION_GRAPH` + A8 + A10 | A7 action graph 基于动作分类模型与采样帧 |
| `LongViTU` | hierarchical/event tree | A5 + A7 `EVENT_TREE` + A8 + A9 | A7 event tree 是当前实现中的近似结构 |
| `Pinpointing Trigger Moment for Grounded Video QA` | QA-first | A8 `QA_FIRST_GROUNDING` | 需要外部 QA 草稿与可匹配的 temporal/segment evidence |
| `Video-in-the-Loop` | 推理旁路 | 不作为当前 A1-A10 主链实现 | 需要额外 question-given localization/re-read 能力 |

## 5. 推荐展示顺序

1. 先展示 A1-A3 的分段、编排、采样。
2. 再展示 A4-A6 的描述、时序、文本证据。
3. 用 A7 展示 `NARRATION_CHUNK`、`EVENT_TREE`、`ACTION_GRAPH` 三种真实模式。
4. 用 A8 展示规则物化或 LLM 物化 QAE。
5. 用 A9/A10 展示 revision record 和 quality record 如何出站。
