# 双机协作

## Remote 分工

- `origin`：当前仓库唯一的远端，用于日常提交和双机同步

查看当前 remote：

```powershell
git remote -v
```

推荐结果：

```text
origin   https://github.com/Wenwen555/video-modifier-system.git
```

## 首次配置

第一台电脑在仓库根目录执行：

```powershell
.\tools\setup_git_dual_machine.ps1
```

如果还没配好 `origin`，也可以直接传当前仓库网址。脚本会自动把网页地址规范化成 Git 地址：

```powershell
.\tools\setup_git_dual_machine.ps1 `
  -OriginUrl "https://github.com/Wenwen555/video-modifier-system/tree/main"
```

第二台电脑建议直接从当前仓库克隆：

```powershell
git clone https://github.com/Wenwen555/video-modifier-system.git
cd video-modifier-system
.\tools\setup_git_dual_machine.ps1
```

## 日常同步

开始工作前：

```powershell
git sync-project
```

结束工作后：

```powershell
.\tools\git_sync_project.ps1 -CommitMessage "checkpoint: current work"
```

脚本会自动执行：

1. `git add -A`
2. `git commit -m "..."`
3. `git fetch origin 当前分支`
4. `git pull --rebase origin 当前分支`
5. `git push origin 当前分支`

如果当前分支第一次推送到远端，脚本会自动改用 `git push -u origin 当前分支`。

## 两台电脑切换

在电脑 A 推送后，到电脑 B 只需要：

```powershell
git sync-project
```

然后继续编辑即可。建议不要在两台电脑上同时修改同一份核心文件后再一起推送。
